# Movie Theater
## Flask RESTful APP

### Installation:
1. Install [Python](https://www.python.org/downloads/)
1. Install [Node](https://nodejs.org/en/)
1. Install Gulp 
    ```bash
    npm install gulp-cli -g
    npm install gulp -D
    ```
1. Install the requirments
    ``` bash
    pip install Flask
    pip install Flask-RESTful
    pip install Py-Mongo==2.8
    ```
